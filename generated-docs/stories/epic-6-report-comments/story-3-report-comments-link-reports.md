# Story: Report Comments - Link to Reports

**Epic:** Report Comments
**Story:** 3 of 4

## User Story
**As an** Analyst, **I want** to link comments to specific reports **So that** commentary is associated with the correct report

## Acceptance Criteria
- [ ] Given I am adding/editing a comment, when I select ReportListId, then I see a dropdown of all available reports for the current batch
- [ ] Given I save a comment with a ReportListId, when the save completes, then the ReportListName is displayed in the grid
- [ ] Given a report has multiple comments, when I view the grid, then all comments for that report are grouped together

## API Endpoints
Report list is retrieved from configuration or a dedicated endpoint (not specified in the provided API docs, may need clarification).
