# Epic 1: Foundation & Start Page

## Description

This epic establishes the foundational structure of the InvestInsight application, including the home page setup, navigation framework, and the Start Page that serves as the operational hub for creating and monitoring report batches. Users will be able to initiate new monthly report cycles, view current batch status, access key screens, and navigate the application efficiently.
